# Placeholder Google OAuth (Web client ID) for app-frontend
#
# Put NEXT_PUBLIC_GOOGLE_CLIENT_ID in Secret `frontend-config` (Helm / K8s). The root layout
# injects it at runtime as window.__GOOGLE_CLIENT_ID__ so you do NOT need to rebuild the image
# when only this value changes — restart the frontend pod after patching the secret.
#
# Patch example (namespace dev):
#   kubectl patch secret frontend-config -n dev --type merge -p \
#     '{"stringData":{"NEXT_PUBLIC_GOOGLE_CLIENT_ID":"YOUR_ID.apps.googleusercontent.com"}}'
#   kubectl rollout restart deployment/frontend -n dev
#
# Vault: add the same key to KV path secret/data/dev/frontend-config (synced to Secret frontend-config).
